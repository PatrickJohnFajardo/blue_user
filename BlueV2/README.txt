==========================================
BLUE V2 - BACCARAT AUTOMATION BOT
==========================================

Welcome to Blue V2. This folder contains everything you need to run the bot.

QUICK START:
1. Double-click 'Install_Bot.bat'.
2. If Tesseract OCR is not installed, follow the prompts to install it.
3. A shortcut named 'Blue' will appear on your Desktop.
4. Run the bot from your Desktop.

IMPORTANT RULES:
- Screen Scaling: Set your Windows display scaling to 100%.
- Calibration: You MUST run calibration (Menu > Run Calibration) the first time you use the bot.
- Password: The calibration password is 'akhlys11'.

FILES IN THIS FOLDER:
- Blue.exe: The main application (the "brain").
- Data/: Contains your settings.
- Setup/: Contains the required OCR engine.
- Logs/: Where the bot keeps track of its actions.

DO NOT DELETE OR MOVE ANY FILES OUTSIDE THIS FOLDER.
